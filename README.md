# EuroSplit - Plan an event and split the expenses!

### This project was developed for the college unit Object Oriented Programming.

### Grade: 10/10

## Features:

- Create a user
- Create a split
- Select a split
- Create an expense
- Check the balance

## Technologies used:

- Java

## Tools

- Maven

## Commands:

- login
- new split
- select split
- new expense
- balance
- quit

## Side Notes:

All the data is stored locally.
